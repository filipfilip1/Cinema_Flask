# Cinema_Flask

## Główne technologie:
Python, Flask, SQLAlchemy, SQLite, JavaScript, CSS, HTML

## Funkcje:
- Przeglądanie miast oraz listy kin w danym mieście
- Wybór filmu i seansu
- Rezerwacja biletów z wyborem miejsc na sali kinowej
- Potwierdzenie rezerwacji
